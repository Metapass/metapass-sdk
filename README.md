# Metapass SDK

A SDK for Metapass written in typescript for interacting with Metapass Protocol on solana.

## Installation

Install metapass-sdk with npm

```bash
  npm install metapass-sdk 
```

## Usage/Examples

```javascript
import { PROGRAM_ID } from "metapass-sdk";
```
